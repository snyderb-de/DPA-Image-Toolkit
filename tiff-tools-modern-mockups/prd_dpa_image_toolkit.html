# Product Requirements Document: DPA Image Toolkit ## 1. Project Overview **DPA Image Toolkit** is a specialized desktop application for archive staff, focused on streamlining image processing workflows like cropping, merging, and splitting TIFF files. The app emphasizes professional, archival-inspired design with a modern, "glassy" aesthetic that feels reliable and premium. ## 2. Core Functionality & User Flows The application is organized around four primary tools, accessible from a central Home grid and a consistent left sidebar. ### 2.1. Navigation & App Shell - **Left Sidebar:** Provides constant access to the Home screen and the four main tools. Includes branding and user profile/settings. - **Main Content Area:** The primary workspace for tool interaction. - **Bottom Status/Progress Bar:** Persistent area for real-time updates on processing tasks, system status, and notifications. ### 2.2. Home Screen - **The Launchpad:** A 2x2 grid of cards representing the four tools. - **Card Design:** Each card includes a clear icon/emoji, tool name, brief description, and a subtle "accent" treatment unique to the tool. - **Aesthetic:** Polished and welcoming, with staggered entry animations for the cards. ### 2.3. Tool Screens (General Pattern) - **Drop Zone / File List:** Large, intuitive area for adding images or folders. - **Configuration Panel:** Settings specific to the tool (e.g., crop margins, output paths). - **Preview Area:** Real-time visual feedback of changes (where applicable). - **Action Button:** Prominent, clearly labeled button to start processing (e.g., "Start Auto Crop"). ### 2.4. Specific Tools - **Auto Crop:** Detects and removes empty space around scanned objects. - **Merge TIFFs:** Combines multiple single-page TIFFs into a single multi-page file. - **Split Multi-Page TIFFs:** Breaks a single multi-page file into individual image files. - **Add Border:** Adds standardized archival-style borders/margins to images. ## 3. Design Principles - **Archival Modernity:** Combining classic museum/library cues (parchment, steel, slate) with modern UI techniques (glassmorphism, layering). - **Trustworthy & Calm:** Avoids flashy colors and hyper-active animations. Focuses on clarity, readability, and a sense of "careful preservation." - **Editorial Typography:** Clear hierarchy with sophisticated font choices that feel like a curated catalog. - **Dual-Theme Excellence:** Both Dark (Lab) and Light (Museum) modes must be highly accessible and visually distinct but cohesive. ## 4. Visual Elements & Accents - **Color Palette:** Warm amber (preservation), brass (hardware), slate/charcoal (digital precision), and parchment/vellum (paper history). - **Glassmorphism:** Frosted glass panels over dark slate or light paper backgrounds to create depth and focus. - **Subtle Cues:** Minimalistic references to film edges, catalog cards, and archival boxes in the UI details. ## 5. Accessibility & UX - High contrast for primary text. - Large, clickable targets for tool cards and primary buttons. - Redundant signaling (color + icons) for status and progress. - Consistent visual language across all tools to minimize cognitive load.
